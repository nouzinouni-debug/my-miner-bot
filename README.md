# Telegram Mining Bot

هذا المشروع عبارة عن بوت تليجرام للتحكم في عملية التعدين على السيرفرات السحابية.

## المميزات
- التحكم الكامل في التعدين عبر أوامر تليجرام.
- يعمل في الخلفية على السيرفرات السحابية (Railway/Replit).
- لا يستهلك موارد هاتفك.

## الأوامر المتاحة
- `/start` : لبدء استخدام البوت.
- `/mine` : لتشغيل عملية التعدين.
- `/stop` : لإيقاف عملية التعدين.

## المتطلبات
- [Railway.app](https://railway.app) أو أي سيرفر سحابي.
- Bot Token من @BotFather.
